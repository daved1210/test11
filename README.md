# 待办事项 (Todo)

一个现代化的待办事项管理应用，支持中英文切换和明暗主题。

## 功能特点

- **优先级管理**：支持高、中、低三种优先级，任务按优先级排序
- **双语支持**：支持简体中文和 English，随时切换
- **主题切换**：支持浅色、深色和自动（跟随系统）三种模式
- **本地存储**：任务数据保存在浏览器本地存储中，刷新页面不丢失
- **流畅动画**：精心设计的交互动画，提升用户体验
- **响应式设计**：适配桌面端和移动端设备

## 技术栈

- 纯原生 HTML、CSS、JavaScript
- Google Fonts（Noto Sans SC、JetBrains Mono、Bebas Neue）
- LocalStorage 数据持久化

## 使用说明

1. 直接打开 `index.html` 文件即可使用
2. 在输入框中输入任务内容，点击"添加"或按 Enter 键添加任务
3. 点击任务左侧的复选框标记任务完成
4. 点击优先级按钮（高/中/低）选择新任务的优先级
5. 鼠标悬停在任务上显示编辑和删除按钮
6. 点击右上角的"中/EN"切换语言
7. 点击右上角的太阳/月亮图标切换主题
